# حقيقة القومية العربية — Arab Nationalism: Truth & Myth

Interactive trilingual web app based on **"Haqiqat al-Qawmiyah al-Arabiyah wa Usturat al-Ba'th al-Arabi"** by Sheikh Mohammed al-Ghazali (1917-1996).

## Features

- 20 knowledge cards on Arab identity, Islam vs nationalism, Ba'ath critique, and unity through Islam
- Critique section analyzing nationalist ideology
- Daily habits tracker
- Interactive quiz
- Narrator with karaoke, speed/pitch control, sleep timer
- Three themes (Charcoal, Night, Slate)
- Three languages (Arabic, English, French)
- Dua panel and daily wisdom

## Tech

Pure HTML/CSS/JS — no frameworks, no build step. Works offline as a PWA.
